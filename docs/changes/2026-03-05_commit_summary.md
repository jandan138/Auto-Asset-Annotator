## Notes

1. **No Breaking Changes:** All changes are backward compatible. Existing annotation files can be processed by the batch fix script.

2. **Normalization Logic:** The regex pattern `(\d+\.?\d*)` extracts the first numeric value from a string, which handles cases like:
   - "0.5 kg" -> "0.5"
   - "0.30 * 0.30 * 0.05 meters" -> "0.30 * 0.30 * 0.05"
   - "Estimated mass is 0.1 kg." -> "0.1"

3. **Category Override:** The category is now always set from the directory name in the asset path, ensuring consistency with the file system structure.

4. **Batch Fix Script:** Can be run on the existing 49,808 annotation files to normalize them without re-running the VLM inference.
